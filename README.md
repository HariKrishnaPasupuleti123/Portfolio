# Modern Developer Portfolio

A modern and responsive personal portfolio website built to showcase my projects, technical skills, and achievements.

## 🚀 Live Demo
- Local: [http://localhost:5173](http://localhost:5173)

## ✨ Key Features

- **🔥 Firebase Studio Aesthetic**: A sophisticated dark theme with glowing orange-red accents and interactive glassmorphism components.
- **🤖 AI-Assisted Development**: A featured section showcasing practical hands-on experience with modern AI tools like **Antigravity**, **GitHub Copilot**, **ChatGPT**, and **Claude**.
- **📈 Professional Timeline**: Detailed internship experiences at **Rupeek Fintech**, **TAP Academy**, and **AICTE Edunet**.
- **💼 Project Gallery**:
    - **🔐 Decentralized Certificate Issuance**: Blockchain-based security system.
    - **📋 Project Management System**: Full-stack Spring Boot & MongoDB backend.
    - **🕵️ Image Steganography**: Computer Vision based covert encryption using OpenCV.
    - **🎮 Tic-Tac-Toe AI**: Java-based game applying high-level OOP design patterns.
- **📱 Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Vanilla CSS (Global variables, Flex/Grid, Keyframe animations)
- **Deployment**: Optimized for rapid deployment and Vercel/Netlify compatibility.
